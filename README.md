# 🔢 JS Largest and Smallest Number Check

A simple JavaScript project that finds the **largest** and **smallest** number among three input values using conditional statements (`if...else`).

---

## 📌 Project Overview

This project demonstrates how to:

- Accept three numbers from the user
- Compare the numbers using JavaScript
- Find the **largest** number
- Find the **smallest** number
- Display the results dynamically

It is a beginner-friendly JavaScript project that helps you understand decision-making with conditional statements.

---

## ✨ Features

- ✅ Takes three user inputs
- ✅ Finds the largest number
- ✅ Finds the smallest number
- ✅ Uses `if...else` conditions
- ✅ Beginner-friendly project
- ✅ Simple and clean code
